# Types

## Description

`types.h` is a header defined in `std/types.h` for every `types` _typedef_ or _macros_ used by the **YutsuOS C standard**.

| Typename | Definition         |
| -------- | ------------------ |
| u8       | unsigned char      |
| u16      | unsigned short     |
| u32      | unsigned int       |
| u64      | unsigned long long |
| uptr     | unsigned long int  |
| i8       | signed char        |
| i16      | signed short       |
| i32      | signed int         |
| i64      | signed long long   |
| f32      | float              |
| f64      | double             |
| bool     | \_Bool             |
| true     | 1                  |
| false    | 0                  |
| NULL     | ((void \*)0)       |
